= 更改 Deploy 管理员密码

您可以根据需要使用命令行界面更改 Deploy 虚拟机管理员帐户的密码。

.步骤
. 使用管理员帐户Sign in到 Deploy 实用程序 CLI。
. 更改密码：
`password modify`
. 根据您的环境对所有提示做出适当的响应。
