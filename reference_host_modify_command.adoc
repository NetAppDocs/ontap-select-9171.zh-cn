= 修改ONTAP Select Deploy 的主机管理服务器
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
你可以使用 `host modify`使用ONTAP Select Deploy 实例修改主机管理服务器的命令。



== 语法

`host modify [-help] [-foreground] -name _name_ -mgmt-server _management_server_ [-username _username_]`



== 必需参数

[cols="35,65"]
|===


| 参数 | 描述 


 a| 
`-name _name_`
 a| 
您要修改的主机的 IP 地址或 FQDN。



 a| 
`-mgmt-server _management_server_`
 a| 
要设置为主机的主机管理服务器的 IP 地址或 FQDN。指定“-”（连字符）可取消设置主机的管理服务器。必须先添加此管理服务器的凭据，然后才能使用以下方式注册此主机： `credential add`命令。

|===


== 可选参数

[cols="35,65"]
|===


| 参数 | 描述 


 a| 
`-help`
 a| 
显示帮助信息。



 a| 
`-foreground`
 a| 
此参数控制长时间运行的命令的行为。如果设置，该命令将在前台运行，并且与操作相关的事件消息将在发生时显示。



 a| 
`-username _username_`
 a| 
有权访问此主机的用户名。仅当主机不受管理服务器管理（即由 vCenter 管理的 ESX 主机）时才需要此用户名。

|===
